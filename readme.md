## Installation

### Linux

1. Clone the source locally:

```sh
$ git clone https://github.com/montecamo/splashy
$ cd frontend
```

2. Install project dependencies:

```sh
$ npm install
```

3. Start the application:

```sh
$ npm run serve
```
